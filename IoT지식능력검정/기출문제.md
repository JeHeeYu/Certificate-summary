### 아래의 내용은 사물인터넷 플랫폼 기술 중, 어떤 기술을 설명한 것인가?
> 정답률 : 55%

```
- 클라이언트-서버방식과 P2P(Peer-To-Peer) 방식으로 구분된다.
- oneM2M 표준과 관련있다.
- 사물인터넷 서비스 생태계의 핵심적 역할을 수행한다.
```

1. 식별체계 기술
2. 검색 기술
3. 서비스 컴포지션 기술
4. 시맨틱 기술


<details>
<summary><h3>정답</h3></summary>

### 2

<문제 해설>
<br>
사물인터넷 플랫폼 기술에는 다음과 같은 기술이 있음
- 식별체계 기술 : 어떤 대상을 유일하게 식별할 수 있는 방법을 제공하는 기술을 식별체계 기술
- 검색 기술 : 사용자가 원하는 서비스를 제공받기 위하여 정보나 리소스를 찾고 찾아진 결과를 쉽게 활용할 수 있도록 제공하는 기술
  - 클라이언트-서버 방식과 P2P 방식으로 구분함
  - 대표적인 예로 클라이언트 서버 방식은 oneM2M 방식을, P2P 방식에는 AllJoyn 방식이 있음
- 사물 가상화 기술 : 물리적 환경에 존재하는 다얗안 사물의 정보를 플랫폼 또는 디바이스에 표현하기 위해 추상화된 형태로 리소스를 생성하는 기술
- 장치관리 기술 : 디바이스의 초기설정, 고장진단, 펌웨어/소프트웨어 다운, 모니터링 등을 위한 기술
- 서비스 컴포지션 기술 : 서비스 지향 구조에서 다양한 서비스를 연동하기 위한 기술
- 시맨틱 기술 : 분산환경에서 리소스에 대한 정보와 리소스들의 관계-의미 정보를 기게가 처리할 수 있도록 온톨로지 형태로 표현하고 이를자동화된 기계가 처리하도록 하는 프레임워크 기술
