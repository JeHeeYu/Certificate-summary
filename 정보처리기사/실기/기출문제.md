- 프로젝트 개발 기간 :
- 계산식 :

<details>
<summary><h3>정답</h3></summary>

### 20개월 / (30,000 라인 / 300라인) / 5명 = 20개월

<문제 해설>
<br>
- Man Month에 프로젝트 인원과 관련 없이 한 사람이 프로젝트를 할 때 걸리는 시간으로 Man Moth = LoC / 프로그래머의 월간 생산성 = 30,000 / 300 = 100개월
- 프로젝트 기간 = Man Moth / 프로젝트 인력 = 100 / 5 = 20
